1.
root: 沒有父節點的節點

2.
degree:一個節點的子節點數

3. 
non-leaf(非葉子節點):有子節點的節點.
leaf(葉子節點):沒有子節點的節點.

3.
level: 絕對位址
root的level為1

距離,不算當下的位置 開始計算, 跳到目的地的count;
a  b  c  d  e  f ; a到f的距離, 不算a, 算b,c,d,e,f,總共5次count=> 5為a到f的距離

4.
Depth: 最大的level數



(*)pre-order, inorder, post order

(*)complete binary tree

(*)full binary tree
